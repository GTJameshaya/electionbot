\# Бот для сбора показаний счетчиков



Бот для сбора показаний счетчиков по двум домам: Волкова д.9 к.1 и Химиков 4.



\## Функционал

\- Выбор дома

\- Ввод показаний квартир

\- Отправка файла в группу

\- Автоматическое архивирование



\## Установка

1\. Клонируйте репозиторий

2\. Установите зависимости: `pip install -r requirements.txt`

3\. Создайте файл `.env` с вашими токенами

4\. Запустите: `python bot.py`



\## Деплой на Bothost

Платформа автоматически запустит бота после загрузки файлов.
